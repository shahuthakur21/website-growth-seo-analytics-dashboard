# Website Growth & SEO Analytics Dashboard

This project focuses on analyzing **website traffic, user engagement, SEO performance, and conversion behavior** to identify practical growth opportunities. It was built as an end-to-end analytics project using **PostgreSQL, SQL, Power BI, and DAX**, with the goal of turning raw website data into actionable business decisions.

Instead of only creating charts, the project follows a real analyst workflow: storing data in a database, writing SQL queries to uncover patterns, and then building an interactive dashboard to present insights clearly.

---

## 🚀 Project Objective

The main objective of this project is to understand:

* Which traffic channels drive the most users
* Which sources convert the best
* Whether mobile users face UX issues
* How strong user retention is
* Where the conversion funnel can be improved

This makes the project highly relevant for **growth analytics, SEO analysis, digital marketing, and user behavior analysis**.

---

## 🛠️ Tools Used

* **PostgreSQL** – data storage
* **SQL** – business queries and performance analysis
* **Power BI** – interactive dashboard and reporting
* **DAX** – KPI measures and percentage calculations
* **Excel / CSV** – dataset preparation

---

## 📊 Key KPIs Tracked

The dashboard includes the following business KPIs:

* Total Users
* Total Sessions
* Conversion Rate (%)
* Bounce Rate (%)
* Retention Rate (%)
* Average Session Duration

---

## 📈 Analysis Performed

The SQL analysis was focused on:

* Traffic source performance
* Device-based bounce and conversion comparison
* Daily growth trends
* New vs returning users
* Conversion funnel analysis
* Poor performing channel identification
* Retention analysis

---

## 💡 Key Insights

Some major insights identified from the project:

* **Organic traffic** drives the highest user volume
* **Direct traffic** shows the best conversion performance
* **Mobile users** have higher bounce rates than desktop users
* **Social traffic** underperforms in conversion efficiency
* Retention ratio suggests scope for stronger re-engagement strategies

---

## 🎯 Business Recommendations

Based on the analysis, the following growth recommendations were proposed:

* Improve **mobile UX and page speed**
* Optimize **social campaign targeting**
* Strengthen **SEO content for intent-based traffic**
* Use **email and personalized recommendations** for retention
* Improve **CTA placement and landing page flow**
